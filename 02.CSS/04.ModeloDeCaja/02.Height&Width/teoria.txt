Existen varias propiedades en CSS que se pueden utilizar para controlar el tamaño de un elemento:

* height: Controla la altura de un elemento sin contar padding, border o margin.

* min-height & max-height: Controlan la altura mínima y máxima de un elemento. Es útil cuando en height tenemos un valor relativo
como puede ser porcentaje, ya que si no definimos un alto mínimo y máximo, al cambiar el valor de height el elemento puede quedar
muy pequeño o muy grande.

* width: Controla el ancho de un elemento sin contar padding, border o margin.

* min-width & max-width: Controlan la anchura mínima y máxima de un elemento. Es útil cuando en width tenemos un valor relativo
como puede ser porcentaje, ya que si no definimos un ancho mínimo y máximo, al cambiar el valor de width el elemento puede quedar
muy pequeño o muy grande.
